# Lumen Analytics Website

Static multi-page website for Lumen Analytics, an analytics and practical AI solutions consultancy.

## Pages

- `index.html` - home page
- `what-i-solve.html` - data, strategy, and AI solution positioning
- `services.html` - service offerings
- `sales-tool.html` - sales analytics tool overview
- `ai-agents.html` - workflow triage agent demo
- `how-it-works.html` - process
- `about.html` - about section
- `pricing.html` - pricing packages
- `sales_summary_tool/` - browser-only sales summary upload tool

## Local Preview

```bash
python3 -m http.server 8520
```

Then open:

```text
http://localhost:8520
```

## Notes

The site is static HTML/CSS/JavaScript and can be served directly by GitHub Pages.
